# Annals of Surgical Oncology — Submission Guidelines

## Original Article Requirements

| Item | Requirement | Our Status |
|------|-------------|:----------:|
| Abstract | Structured, <=250 words | PASS |
| Body Text | ~3,000-5,000 words | PASS |
| References | No strict limit | PASS (16) |
| Figures/Tables | No strict limit | PASS (6 main + suppl) |
| Cover Letter | Required | PASS |
| STROBE | Required for observational | PASS (22/22) |
| Ethics Statement | Required | PASS |
| COI Statement | Required | PASS |
| Data Availability | Required | PASS |

## Submission Workflow

1. Register at [Editorial Manager](https://www.editorialmanager.com/aso/)
2. Upload DOCX + PDF + Cover Letter
3. Enter author info, suggested reviewers
4. Submit -> Desk Editor -> Peer Review -> Decision (~3-4 weeks)
