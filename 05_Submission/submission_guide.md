# Annals of Surgical Oncology — 投稿指南

## 原始研究文章要求

| 项目 | 要求 | 我们现状 | 状态 |
|------|------|----------|:--:|
| 摘要 | 结构化 ≤250词 | 250 words | ✅ |
| 正文 | ~3,000-5,000 词 | ~3,400 words | ✅ |
| 参考文献 | 无上限 | 16 | ✅ |
| 图表 | 无严格限制 | 6主图 + 补充 | ✅ |
| Cover Letter | 必须 | ✅ | ✅ |
| STROBE | 观察性研究必须 | ✅ 22/22 | ✅ |
| 伦理声明 | 必须 | ✅ | ✅ |
| COI声明 | 必须 | ✅ | ✅ |
| Data Availability | 必须 | ✅ | ✅ |

## 投稿流程

1. 注册 [Editorial Manager](https://www.editorialmanager.com/aso/)
2. 上传 DOCX + PDF + Cover Letter
3. 填写作者信息、建议审稿人
4. 提交 → Desk Editor 初审 → 送审 → 3-4周收审稿意见
